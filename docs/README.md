# Cost Saver - GitHub Pages Documentation

This folder contains the static HTML pages for Cost Saver's Privacy Policy and Terms of Service, hosted on GitHub Pages.

## GitHub Pages Configuration

- **Base URL:** `https://Vinaysparkstackdigital.github.io/costco-saver-pro/`
- **Privacy Policy:** `https://Vinaysparkstackdigital.github.io/costco-saver-pro/privacy`
- **Terms of Service:** `https://Vinaysparkstackdigital.github.io/costco-saver-pro/terms`
- **Index/Documentation:** `https://Vinaysparkstackdigital.github.io/costco-saver-pro/`

## Files

- **privacy.html** - Privacy Policy with app permissions, data collection, and user rights
- **terms.html** - Terms of Service with legal disclaimers and usage restrictions
- **index.html** - Main documentation page with links to policies
- **_config.yml** - GitHub Pages Jekyll configuration

## Setup Instructions

### Enabling GitHub Pages

1. Go to repository settings: `https://github.com/Vinaysparkstackdigital/costco-saver-pro/settings/pages`
2. Select **Source**: Deploy from a branch
3. Select **Branch**: `main` (or `feature/my-work`)
4. Select **Folder**: `/docs`
5. Click **Save**

The pages will be available at: `https://Vinaysparkstackdigital.github.io/costco-saver-pro/`

## Updating Policies

1. Edit the `.html` files in the `/docs` folder
2. Update the `Last Updated` date
3. Commit and push changes
4. GitHub Pages will automatically rebuild the site

## Mobile App Integration

The mobile app links to these GitHub Pages URLs with `target="_blank"` to open in a web browser or external app. This ensures:
- ✅ Always the latest policy version
- ✅ No version mismatches
- ✅ Easy updates without app re-deployment
- ✅ Professional hosting on GitHub infrastructure
